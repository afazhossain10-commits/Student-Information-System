# Student Information System

## 📌 Overview
This is a Java-based Student Management System.

## ⚙️ Features
- Add Student
- View Students
- Search Student
- Update Student

## ▶️ How to Run
1. Compile all Java files
2. Run StudentInformationSystem.java

## 📁 Project Structure
- src/ → Source code
- docs/ → Documentation
- test_data/ → Sample data
